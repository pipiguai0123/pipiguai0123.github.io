四、实现某些操作

1、实现：通过连接公网服务器（192.168.94.156）的3008端口远程连接内网机器（192.168.94.154）

1.1、完成 步骤一、二、三后，点击隧道 >> 点击新增

1.2、输入我们设置的服务端端口和目标ip端口 >> 新增

1.3、在我们的公网机器上使用xshell连接3008端口

注：出现如下内容即实现了连接公网3008端口，远程连接内网机器！即：可以在其他机器上通过连接公网机器+设置的服务端口进行连接内网机器

2、实现：使用sock模式进行内网访问，设置认证账号：admin 认证密码：admin123 认证端口为3007！

2.1、完成 步骤一、二、三，在步骤三新增客户端时>> 填写basic认证用户名和密码 >> 新增

2.2、点击隧道 >> 新增 >> 选择模式为SOCKS代理 >> 填写服务端端口

2.3、下载并安装Proxifier。

下载地址：http://soft.onlinedown.net/soft/971579.htm

#中文汉化版，直接安装。

2.4、打开Proxifier >> 点击配置文件 >> 代理服务器

2.5、添加代理服务器 >> 输入公网ip（192.168.94.156）端口：3007 >> 协议选择socks版本5 >> 验证选择启用：（输入设置的basic用户名：admin 密码： admin123） >> 点击检查

2.6、显示通过以后 >> 连接

2.7、连接后 >> 点击代理规则，规则名称默认选择的为Localhost，或者自己新增规则名称（自己新增的话，可以根据自己的需求选择由哪个应用程序访问网络，我这边选择用qq浏览器访问该网络） >> 动作选择为刚刚我们socks的ip >> 点击确定

2.8、确定以后，就可以让在另一个地方的机器（不同网络下的机器）ping该内网机器ip（192.168.94.154）是否通，或者ssh连接该内网ip（192.168.94.154），或者直接访问该内网机器某些运行的服务。

注：内网内其他的机器能返回数据，即已成功实现使用sock模式进行内网访问！
