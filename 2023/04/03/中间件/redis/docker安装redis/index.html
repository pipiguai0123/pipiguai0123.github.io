上传conf文件

这里可能还需配置bind, protected-mode

  • protected-mode 是在没有显示定义 bind 地址（即监听全网断），又没有设置密码 requirepass时，protected-mode 只允许本地回环 127.0.0.1 访问。也就是说当开启了 protected-mode 时，如果你既没有显示的定义了 bind 监听的地址，同时又没有设置 auth 密码。那你只能通过 127.0.0.1 来访问 redis 服务。

  • 关闭绑定[允许所有外网访问]： # bind 127.0.0.1 或者 解除只有本地能链接客户端 0.0.0.0 -::1

  • 开启密码验证： requirepass redispassword

  • 开启数据持久化[aof]：appendonly yes

  • 配置日志存放目录和文件：logfile /logs/redis.log
