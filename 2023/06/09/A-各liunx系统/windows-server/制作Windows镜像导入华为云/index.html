制作windows镜像导入华为云

一、安装前：制作镜像时，建议使用原版纯净镜像

镜像下载地址（国内）：https://hellowindows.cn/

二、系统安装后操作：

1、开启Administrator账户登录

参考文档：https://www.cnblogs.com/pam-sh/p/12239172.html

1.1、以管理员身份打开Windows-PowerShell

1.2、在命令提示符中输入命令“net user administrator /active:yes”后回车

1.3、administrator管理员账户已开启，点击任务栏即可切换

1.4、删除普通账户、使用administrator登录

2、开启连接远程

设置>-系统>-远程桌面=启用远程桌面

三、开启ping（视情况而定，windows默认不开启）

高级安全windows防火墙–》入站规则–》文件和打印机共享（回显请求-ICMPv4-In）–》属性–作用域（选择任何IP）–》启用规则

四、安装华为云相关依赖插件

前言参考文档：https://support.huaweicloud.com/usermanual-ims/ims_01_0204.html

1.安装并配置Cloudbase-Init工具

参考链接：https://support.huaweicloud.com/usermanual-ims/zh-cn_topic_0030730602.html

2.安装驱动UVP VMTools

参考链接：https://support.huaweicloud.com/usermanual-ims/ims_01_0319.html

https://ecs-instance-driver.obs.cn-north-1.myhuaweicloud.com/vmtools-windows.zip
