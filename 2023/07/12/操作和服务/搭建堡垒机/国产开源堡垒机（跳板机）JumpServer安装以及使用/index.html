国产开源堡垒机（跳板机）JumpServer安装以及使用

一、安装JumpServer

因Jumpserver需要使用redis和数据库，建议使用外置 数据库 和 Redis

DB Version Cache Version
MySQL >= 5.7 Redis >= 5.0
MariaDB >= 10.2

1、在服务器有网的情况下可采用《一键部署》方式

默认会安装到 /opt/jumpserver-installer-v2.20.2 目录

备注：如果需要安装2.16.2版本，只需要将curl命令中的v2.20.2替换为v2.16.2

2、安装完成后会生成配置文件 /opt/jumpserver/config/config.txt

3、修改配置文件后，因安装的不同的版本会生成不同的版本文件夹

启动后>>访问本机ip+80端口，初始账号和密码都为：admin

备注：官方文档上有
